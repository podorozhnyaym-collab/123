# Сайт про собаку — "Рекс"

Це простий статичний сайт українською мовою про собаку (демонстраційний). Він складається з HTML, CSS і невеликого JS для інтерактиву (тема, галерея, форма).

Файли:
- index.html — головна сторінка
- styles.css — стилі
- script.js — мінімальна логіка (тема, галерея, форма)
- README.md — цей файл

Як запускати локально:
1. Скопіюй файли в одну папку.
2. Відкрий `index.html` у браузері (подвійний клік або через `File -> Open`).
3. Для локального server (рекомендовано), можна запустити простий HTTP-сервер:
   - Python 3: `python -m http.server 8000` і відкрити http://localhost:8000

Деплой на GitHub Pages:
1. Додай файли в репозиторій (наприклад, в гілку `main`).
2. У налаштуваннях репозиторію (Settings → Pages) вкажи гілку `main` та папку `/` (root).
3. Зачекай кілька хвилин — сайт буде доступний за URL вида: `https://<your-username>.github.io/<repo>`.
